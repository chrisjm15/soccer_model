## What this module does

It takes historical match data and computes **rolling EMA (Exponential Moving Average) attack and defence ratings** for every team in every league. These ratings are used later to predict match outcomes.

Each team has four ratings (all in xG-per-match units):
- `attack_home` — how many xG the team typically generates when playing at home
- `defence_home` — how many xG the team typically concedes when playing at home
- `attack_away` — how many xG the team typically generates when playing away
- `defence_away` — how many xG the team typically concedes when playing away

## EMA logic

The EMA update formula is:
```
new_rating = alpha * observed_value + (1 - alpha) * previous_rating
```

Where `alpha = 0.1` (configurable via a constant at the top of the file).

**For each match:**
- Update the **home team's** `attack_home` EMA using `home_xg` (what they scored at home)
- Update the **home team's** `defence_home` EMA using `away_xg` (what they conceded at home)
- Update the **away team's** `attack_away` EMA using `away_xg` (what they scored away)
- Update the **away team's** `defence_away` EMA using `home_xg` (what they conceded away)

**IMPORTANT: The output row for each match must contain the ratings BEFORE that match is processed.** This prevents lookahead bias. The update happens after saving the pre-match snapshot.

---

## Season transition (regression to mean)

At the start of each new season, each team's ratings are blended toward the league mean:

```
season_start_rating = (1 - SHRINK_WEIGHT) * last_season_final_rating + SHRINK_WEIGHT * league_mean
```

Where `SHRINK_WEIGHT = 0.3` (configurable constant).

**League mean** = average of all teams' final ratings for that rating dimension at end of prior season.

If a team has no prior season data (first appearance in the dataset), initialise all four ratings to the league mean for that season. If there is no prior season at all (very first season), initialise to a sensible prior:
- `attack_home`: 1.60
- `defence_home`: 1.30
- `attack_away`: 1.10
- `defence_away`: 1.50

These are approximate Big 5 league xG averages.
